# Horario de ocupación de aulas EDECA

Página estática lista para GitHub Pages.

## Uso rápido

1. Crear un repositorio en GitHub.
2. Subir el archivo `index.html` a la raíz del repositorio.
3. Activar GitHub Pages desde Settings > Pages.
4. Seleccionar la rama principal y la carpeta raíz.

La página no requiere servidor, base de datos ni librerías externas.

## Datos incluidos

Aulas: L601, L602, L603, 708, 709, 710 y 711.
Periodo: 202620.

Nota: si se agregan logos institucionales oficiales, deben incorporarse como archivos separados, sin modificar su color, proporción ni área de protección.
